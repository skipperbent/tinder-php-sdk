# tinder-php-sdk
Easy to use PHP SDK for accessing Tinder data.

## Installation
Add the latest version of tinder-php-sdk to your ```composer.json```

```json
{
    "require": {
        "pecee/tinder-php-sdk": "1.*"
    },
    "require-dev": {
        "pecee/tinder-php-sdk": "1.*"
    }
}
```
